# 项目简介

## 核心功能
- 系统首页管理
- 用户管理
- 文章分类管理
- 文章信息管理
- 资源类型管理
- 文件资源管理
- 物品分类管理
- 物品信息管理
- 购买信息管理
- 论坛分类管理
- 举报记录管理
- 意见反馈管理
- 敏感词管理
- 论坛交流
- 系统管理
- 个人中心

## 适用场景
- 校园内部交流
- 学生之间信息共享
- 教师与学生互动
- 校园活动通知发布

## 技术特点
- B/S架构，支持Web访问
- MVC设计模式，提高代码结构清晰度
- 使用Java、SpringBoot、Vue等技术栈
- 支持IDEA、JDK1.8、Mysql、Maven、Navcat等开发环境
